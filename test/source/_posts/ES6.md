#### ES6学习笔记

##### <u>1.    let，const命令</u>

​	ES5中只有2个作用域：全局作用域，函数作用域。

​	ES6中增加了块级作用域：{ }包裹的区域。

###### <u>1.1 let</u>

​		只在当前块作用域有效，在外可理解为不存在。

​		let不能重复声明同一个变量。

###### <u>1.2 const</u>

​	声明的时候必须赋值。

​	定义常量，一般声明后不可更改，

​	*特殊情况： 

```javascript
const a = {
    k: 1,
    m: 2
}
a.b = 3
console.log(a) // {k:1, m: 2, b:3} 
//因为对象是引用类型， const存储的是指针，指针一直没有改变，改变的是指针指向的对象
```
